Para ejecutar NumberOfPaths.jar

Abra una consola cmd y navegue hasta esta carpeta y ejecute:

java -jar NumberOfPaths.jar -f input-files/1.txt -sm -t
java -jar NumberOfPaths.jar -f input-files/2.txt -st -t
java -jar NumberOfPaths.jar -d input-files --memoization --time
java -jar NumberOfPaths.jar -d input-files --tabulation --time
